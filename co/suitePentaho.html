Suite décisionnelle Pentaho

Développée en 2004, Pentaho est un logiciel opensource de Business Intelligence (BI)

Pentaho est une suite logicielle qui permet la distribution de fonctionnalités et documents décisionnels à un grand nombre de personnes par l'intermédiaire d'une interface Web.

Pentaho est proposé en version communautaire et en version entreprise soumise à souscription annuelle, avec des modules supplémentaires ainsi qu'un support produit et projet et des garanties de licences.

La suite Pentaho offre différents outils :

  • BI Platform (JBoss Portal)

  • Pentaho Dashboard

  • Report Designer

  • Schema Workbench - Analysis (Mondrian)

  • Data Integration (PDI) (Kettle)

  • Data Mining / Predictive Analysis (Weka)

  • Analytics (Agile BI)

  • IoT Analytics

  • Big Data Integration and Analytics

Méthode : Pentaho Dashboard

Pentaho Dashboard fournit aux utilisateurs professionnels les informations critiques dont ils ont besoin pour comprendre et améliorer les performances organisationnelles.

Fournissez un aperçu immédiat des performances individuelles, sectorielles ou d'entreprise en fournissant des métriques clés dans une interface visuelle attrayante et intuitive.

Méthode : Pentaho Schema Workbench - Analysis

  • Haute performance, analyse interactive de grands ou petits volumes d'informations

  • Exploration "dimensionnelle" des données, par exemple analyse des ventes par ligne de produits, par région, par période de temps

  • Analyse du langage MDX (Multi-Dimensional eXpression) dans SQL (Structured Query Language) pour récupérer les réponses aux requêtes dimensionnelles

  • Requêtes haute vitesse via l'utilisation de tables agrégées dans le SGBDR

Méthode : Pentaho Datamining (Weka)

Weka est un ensemble de classes et d'algorithmes en Java implémentant les principaux algorithmes de data mining : classification, régression, règles d'association rules, clustering, etc.
